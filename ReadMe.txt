# Simple Calculator in C++

This is a basic console calculator written in C++ using `if` statements and `switch` cases.  
It performs four arithmetic operations:

- Addition (+)  
- Subtraction (-)  
- Multiplication (*)  
- Division (/)

## Features

- Simple user interface through the console.
- Uses `if` and `switch` for operation selection.
- Supports double precision numbers.
- Error handling for invalid operators.

## How to Run

Make sure you have a C++ compiler (like `g++`) installed. Then compile and run with:

```bash
g++ calc.cpp -o calculator
./calculator
